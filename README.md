# 🎨 Advanced Drawing App

An advanced drawing application built using HTML, CSS and JavaScript.

## 🚀 Features
- Brush tool for drawing
- Color picker
- Eraser tool
- Clear canvas button
- Smooth drawing experience

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
